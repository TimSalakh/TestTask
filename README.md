# Запуск проекта

## Запуск без Docker

Создайте виртуальное окружение и активируйте его:

```bash
python -m venv venv
source venv/bin/activate  # Для Linux/Mac
venv\Scripts\activate     # Для Windows
```

## Запуск с Docker compose

Создайте виртуальное окружение и активируйте его:

```bash
docker-compose up --build
```
